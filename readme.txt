This plugin will help admin and users of the website to filter the search of page/posts based on some criteria i.e. all the available post types.

== Description ==
1. This plugin will help admin and front end user of the website to filter the search of page/posts based on some criteria i.e. all the available post types. It will save time and avoid going through several unnecessary posts and pages, thus making the task of admin and front end users faster, smoother and smarter.
2. By using this plugin, admin can have search with suggestion of post/page's title or(all the available post types.).
3. This plugin helps users to search out the right post/pages instantly.
4. It saves time and efforts by searching the right post/pages.
5. If admin want to search with suggestion in particular post type then admin must have to check marked on particular post type.
6. On front side admin setting page selection are there. If user wants to display post's excerpt, date, image(featured image) and published date. admin can select criteria and based of these user can filter the pages/posts.
7. User can create more then one search pages with shortcode parameters also. Parameters having post type and limit.


== Installation ==

1. Copy the entire /auto-search-suggestion/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. New Tab called Auto Search will be genereated.
4. Plugin have two settings. Front side settings and Admin side settings.
5. Add shortcode for front side [bt_auto_suggest]  and with parameters [bt_auto_suggest post_type="post,page" limit="3"]
